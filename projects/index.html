EMG Hand Control

Project aims to map signals from Electromyograph (EMG) array on forearm to finger joint angles and visualize the results. Utilizes computer vision as ground truth and a regression neural network to determine joint angles. Uses customized XR hand feature in Unity 6 to visualize both ground truth and prediction

Model Reference Adaptive Control (MRAC) for Drone

Utilizes Python and Webots to compare standalone LQR with MRAC performance on quadrotor motor faliure scenario

Tight Learned Inetrial Odometry (TLIO) Data Loader

Built data pipeline using motion capture system (VICON) and Treadmill (D-Flow) for robot motion profiles (Digit).Utilized the same to develop a data loader in python to reduce IMU (Inertial Measurement Unit) drift by 40 cm/min

Automotive Control Methods

Utilizes Python and Webots to test different control methods (PID, Full State,LQR,EKF + SLAM) on a Tesla car around a set track. Currently in EKF with SLAM state

Electric Vehicle (EV) Battery Management Model

Modelling and Optimization of EV charging and discharging with Vehicle to Home (V2H) capabilities alongside solar panel, HVAC, and stationaary battery systems on a residential scale. Goal is to reduce grid usage and feedback

Power Over Ethernet (POE) Lighting System Implementation

Designed and deployed a Direct Current (DC) lighting system. Determined new system uses 20% less energy on larger scales compared to AC Lighting systems using custom built current data aquicition system based on R-Pi. (Led to my first lead author publication)

Micro Electro Mechanical System (MEMS) mic response and EQ study

Custom Data Power Acquisition System Using Hobby Electronics
